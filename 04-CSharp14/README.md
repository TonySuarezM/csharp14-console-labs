# Nivel 4: Características de C# 14

Estos ejemplos necesitan C# 14 y .NET 10. Exploran `field`, asignaciones con `?.`,
miembros de extensión, mejoras de `Span<T>`, lambdas y miembros parciales.

Prueba a comentar `<LangVersion>14.0</LangVersion>` en `Directory.Build.props` para
comprobar qué sintaxis depende de esta versión, y restaura después el archivo.
