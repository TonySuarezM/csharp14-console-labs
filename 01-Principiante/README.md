# Nivel 1: Principiante

Ejecuta los ejemplos en orden. Practica cambiando mensajes, valores y condiciones.

1. `01-HolaMundo`: estructura mínima y salida.
2. `02-VariablesYTipos`: datos, constantes e interpolación.
3. `03-EntradaYOperadores`: entrada del usuario y cálculos.
4. `04-Condicionales`: decisiones con `if` y `switch`.
5. `05-Bucles`: repetición con `for` y `foreach`.
6. `06-Metodos`: organización y reutilización.
